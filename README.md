# 🌱 Plant Disease Detection

## 📌 Overview
Plant Disease Detection is an AI-powered deep learning project that identifies diseases in plant leaves using image classification. The system helps farmers, students, and researchers quickly detect plant diseases and take preventive actions to improve crop health and productivity.

---

## 🚀 Features

- 🌿 Detects plant leaf diseases from uploaded images
- 🤖 Deep Learning-based image classification
- 📊 High accuracy disease prediction
- ⚡ Fast and user-friendly interface
- 💻 Easy to run locally
- 🌍 Can be extended for real-time agricultural applications

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/plant-disease-detection.git
```

### Move into the project

```bash
cd plant-disease-detection
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

Start the application:

```bash
python app.py
```

Then open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 📸 How It Works

1. Upload a plant leaf image.
2. The image is preprocessed.
3. The trained deep learning model analyzes the image.
4. The system predicts the disease.
5. The predicted disease is displayed to the user.
